# Hprof 文件格式详解

## 概述

Hprof（Heap Profile）是 Java 虚拟机的堆转储文件格式，用于记录 Java 堆内存的详细信息。

## 文件结构

```
+------------------+
|   文件头          |
| (19-25 字节)     |
+------------------+
|   记录 1          |
+------------------+
|   记录 2          |
+------------------+
|      ...         |
+------------------+
|   记录 N          |
+------------------+
```

## 文件头格式

```
+------------------+--------+--------------------------------+
| 字段             | 长度   | 说明                           |
+------------------+--------+--------------------------------+
| Magic String     | 15字节 | "JAVA PROFILE "               |
| Version          | 3-7字节| "1.0.1" 或 "1.0.2"            |
| NULL Byte        | 1字节  | 0x00                          |
| Identifier Size  | 4字节  | 标识符大小(4或8字节)          |
| Timestamp        | 8字节  | 时间戳(毫秒)                  |
+------------------+--------+--------------------------------+
```

## 记录格式

每条记录的结构：

```
+--------------+--------+--------------------------------+
| 字段         | 长度   | 说明                           |
+--------------+--------+--------------------------------+
| Tag          | 1字节  | 记录类型标识                   |
| Timestamp    | 4字节  | 时间戳（毫秒）                 |
| Length       | 4字节  | 数据长度                       |
| Data         | N字节  | 记录数据（Length指定）         |
+--------------+--------+--------------------------------+
```

## 记录类型

| 值 | 名称 | 说明 |
|----|------|------|
| 0x01 | UTF8 | UTF-8 字符串 |
| 0x02 | LOAD_CLASS | 类加载信息 |
| 0x0C | HEAP_DUMP | 堆转储 |
| 0x1C | HEAP_DUMP_SEGMENT | 堆转储片段 |
| 0x2C | HEAP_DUMP_END | 堆转储结束 |
